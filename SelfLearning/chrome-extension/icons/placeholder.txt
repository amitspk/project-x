# Icon Placeholder Files

To use the Chrome extension, you need to create PNG icon files in the following sizes:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels) 
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can:
1. Convert the provided icon.svg to PNG files using an online converter
2. Create your own icons with a design tool
3. Use the extension without icons (Chrome will show a default icon)

The icon.svg file provides a template design with a question mark symbol.
